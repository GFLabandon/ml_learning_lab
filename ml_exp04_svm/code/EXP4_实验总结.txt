📊 实验内容
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

核心目标：对比4种核函数 × 2种多分类策略 = 8种SVM模型组合

核函数对比：
  ✓ Linear（线性核）：最快，但低阶
  ✓ Polynomial（多项式核）：中等复杂度，可调degree参数
  ✓ RBF（径向基函数核）：最灵活，通常最优
  ✓ Sigmoid（sigmoid核）：类神经网络，计算成本高

多分类策略对比：
  ✓ OVR（一对多）：k个分类器，训练快
  ✓ OVO（一对一）：k(k-1)/2个分类器，准确率通常高

数据集：手写数字0-9（10分类）
  • MNIST：70000张28×28灰度图像
  • Digits（备选）：1797张8×8灰度图像

🎯 SVM核函数原理速览
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【基本原理】

SVM寻找最优超平面，使得两类样本间的间隔（margin）最大：

  目标函数：最小化 ||w||²/2 + C Σ ξᵢ
  
  决策函数：f(x) = sign(Σ αᵢyᵢK(xᵢ,x) + b)

其中：
  • w：超平面法向量
  • ξ：松弛变量（允许错误分类）
  • C：正则化参数（权衡准确率与泛化）
  • K(xᵢ,x)：核函数（隐式高维映射）

【四种核函数详解】

1. Linear（线性核）
   ───────────────
   K(x,z) = <x,z>
   
   特点：
   • 数学形式最简单：直接计算内积
   • 计算速度最快：O(n)
   • 参数最少：无额外参数
   
   适用：
   • 数据线性可分或近似线性可分
   • 特征维度高
   • 样本数量超大
   
   例子：文本分类（线性可分）


2. Polynomial（多项式核）
   ──────────────────────
   K(x,z) = (γ<x,z> + r)^d
   
   特点：
   • 通过多项式实现非线性映射
   • 参数：degree(d)、gamma(γ)、coef(r)
   • 计算成本：中等
   
   适用：
   • 中等非线性问题
   • 需要特定多项式阶数的场景
   
   参数调整：
   • d=2：二次多项式，适合大多数问题
   • d=3：三次多项式，更复杂但计算慢
   • d越大，越容易过拟合


3. RBF（径向基函数核）
   ─────────────────────
   K(x,z) = exp(-γ||x-z||²)
   
   特点：
   • 最灵活，可映射到无穷维空间
   • 参数：gamma(γ)
   • 计算成本：高
   
   适用：
   • 高度非线性问题
   • 样本维度不太高时
   • 需要最好准确率时
   
   参数调整（gamma）：
   • γ=0.001：影响范围大，欠拟合风险
   • γ=0.1：影响范围小，容易过拟合
   • γ='scale'（默认）：1/(n_features × X.var())


4. Sigmoid（sigmoid核）
   ──────────────────────
   K(x,z) = tanh(γ<x,z> + r)
   
   特点：
   • 形式类似于神经网络
   • 参数：gamma(γ)、coef(r)
   • 计算成本：高
   
   适用：
   • 特殊的非线性问题
   • 需要与神经网络对应时
   
   注意：
   • 有时不满足Mercer条件（非正定核）
   • 性能往往不如RBF
   • 超参数调优困难


【多分类策略原理】

SVM本身是二分类器，对k分类问题需要采用策略：

1. OVR（One-vs-Rest）
   ──────────────────
   构造方式：
   • 为类别i构建分类器：第i类 vs 其余所有类
   • 共k个分类器
   
   预测过程：
   • 用所有k个分类器预测
   • 选择得分最高的分类器对应的类别
   
   数学形式：
   class = argmax_i(score_i)
   
   优点：
   • 分类器数量少（k个）
   • 训练速度快
   • 易于并行化
   
   缺点：
   • 数据不平衡：第i类 vs 其余(k-1)类
   • 准确率通常较OVO低
   
   复杂度：
   • 对于10分类：10个分类器
   • 训练时间：较快
   • 预测时间：O(k)


2. OVO（One-vs-One）
   ──────────────────
   构造方式：
   • 为每对类别(i,j)构建分类器：第i类 vs 第j类
   • 共C(k,2) = k(k-1)/2个分类器
   
   预测过程（投票机制）：
   • 用所有分类器进行预测
   • 统计每个类别被选中的次数
   • 选择得票最多的类别
   
   数学形式：
   votes_i = Σ [classifier(i,j)(x) == i]
   class = argmax_i(votes_i)
   
   优点：
   • 数据平衡：每个分类器只用两个类别的数据
   • 准确率通常较OVR高
   • 每个分类器训练样本少，收敛快
   
   缺点：
   • 分类器数量多：k(k-1)/2个
   • 训练时间长
   • 预测时间长
   
   复杂度：
   • 对于10分类：45个分类器（10×9/2）
   • 训练时间：长
   • 预测时间：O(k²)

🔧 参数调优建议
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【基本参数】

1. C参数（正则化强度）
   ─────────────────
   • 默认值：1.0
   • 含义：错分样本的惩罚强度
   
   C越大：
   • 越严格约束，容易过拟合
   • 训练时间长
   • 使用：数据量大、噪声少
   
   C越小：
   • 越宽松约束，容易欠拟合
   • 训练时间短
   • 使用：数据量小、噪声多


2. gamma参数（RBF/Sigmoid）
   ───────────────────────
   • 默认值：'scale' (1/(n_features*X.var()))
   • 含义：单个样本的影响范围
   
   gamma越大：
   • 影响范围越小（只看近邻）
   • 容易过拟合
   • 训练慢
   
   gamma越小：
   • 影响范围越大（看全局）
   • 容易欠拟合
   • 训练快


3. degree参数（Poly）
   ──────────────────
   • 默认值：3
   • 含义：多项式的阶数
   
   degree=2：二阶多项式，简单快速
   degree=3：三阶多项式，复杂准确（通常最优）
   degree>=4：高阶多项式，过拟合风险大


【调优流程】

1. 快速搜索
   ────────
   # 先用默认参数找到最优核函数
   for kernel in ['linear', 'poly', 'rbf', 'sigmoid']:
       svm = SVC(kernel=kernel)
       svm.fit(X_train, y_train)
       print(f"{kernel}: {svm.score(X_test, y_test)}")

2. 精细调优
   ────────
   # 对最优核函数进行网格搜索
   from sklearn.model_selection import GridSearchCV
   
   param_grid = {
       'C': [0.1, 1, 10, 100],
       'gamma': [0.0001, 0.001, 0.01, 0.1, 'scale']
   }
   
   grid = GridSearchCV(SVC(kernel='rbf'), param_grid, cv=5)
   grid.fit(X_train, y_train)
   print(grid.best_params_)
   print(grid.best_score_)
